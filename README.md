## Getting Started
### Requirements
- Kotlin 2.0 (with JVM 21)

> [!NOTE]  
> 이전 버전의 JVM 환경에서도 실행 가능할 것으로 추정되나, 검증되지 않았습니다.

### Build

```sh
./gradlew build
```

### Run

```sh
java -jar linguist-1.0-SNAPSHOT-standalone.jar <path>
```
